# EstilosScssProyectoIntegrado
Este repositorio centraliza la arquitectura de estilos SCSS y la guía de maquetación HTML base para nuestros desarrollos. Contiene la definición oficial de la identidad corporativa, incluyendo la paleta de colores completa y los lineamientos visuales, con el objetivo de estandarizar el diseño y facilitar la implementación en futuros proyectos.
